I'm a Program Manager transitioning into Azure engineering. I wanted to build a solid foundation in Azure infrastructure, Bicep deployments, and GitHub Actions CI/CD — and I learn best by building real things.

I scored 661/700 on my first AZ-104 attempt. Close, but not a pass. For the retake I wanted structured hands-on work, not just practice exams. Combining it with Minecraft kept the work grounded in something concrete — every Azure decision had a practical reason behind it.

## Quest Map

Each domain maps to an AZ-104 exam area. The quests below are the original plan. Implementation notes for each are in the sections that follow.

### Domain 1: Manage Azure Identities and Governance (15–20%)
**Quest: The Realm of Identity**
- Set up Entra ID and create users
- Assign RBAC roles for admin access
- Apply resource locks to protect infrastructure
- Connect the server to a custom domain

### Domain 2: Implement and Manage Storage (15–20%)
**Quest: The Storage Vaults**
- Create an Azure Storage Account for world file backups
- Use Blob Storage for server logs and mod files
- Set lifecycle policies to archive old worlds
- Configure SAS tokens for controlled external access

### Domain 3: Deploy and Manage Azure Compute Resources (20–25%)
**Quest: The Virtual Fortress**
- Deploy a VM to host the Minecraft server
- Configure Just-in-Time (JIT) VM access
- Set up auto-shutdown and start schedules for cost control
- Use Custom Script Extension to automate server installation on VM deploy

### Domain 4: Configure and Manage Virtual Networking (20–25%)
**Quest: The Networking Nether**
- Create a VNet and subnets
- Deploy NSGs to restrict traffic to trusted sources
- Configure a public IP and DNS name for the server
- Troubleshoot connectivity using `az network`

### Domain 5: Monitor and Back Up Azure Resources (10–15%)
**Quest: The Watchtower**
- Configure Azure Monitor for CPU and memory tracking
- Set up Log Analytics for event logs and failure detection
- Create alerts for crashes and high resource utilization
- Enable Azure Backup for world files and server data

### Bonus Quests
- Automate server deployment with Bicep
- Build an Azure Portal dashboard for server health
- Set up Azure Bastion for secure VM access
- Use Azure Automation to restart or patch the VM
